# File Templates for Personalize

Reference this file when generating knowledge base files during `/personalize`.

## knowledge/profile.md

```markdown
# Candidate Profile

## Identity
[Name, current role, education, years of experience]

## Core Pillars
1. [Pillar 1 — with evidence from resume]
2. [Pillar 2 — with evidence]
3. [Pillar 3 — with evidence]

## Target Roles
[List of target role types]

## Target Companies
[List if any, or "Open to opportunities"]

## Current Situation
[Context that affects job search strategy]

## Strategic Notes
- Where founder/entrepreneurial experience helps vs needs framing
- Where education is an asset
- Where seniority might be questioned and how to address
- Any career transition narrative needed
```

```markdown
## What This Story Proves
- [Trait 1 it demonstrates]
- [Trait 2]

## Best Used For
- [Question type 1: e.g., "Tell me about a time you led without authority"]
- [Question type 2]
- [Question type 3]

## Where This Helps vs Needs Framing

- Frame carefully: [contexts where it might raise concerns]

## One-Liner Version
[2-sentence version for quick reference in outreach/cover letters]
```
